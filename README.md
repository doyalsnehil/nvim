# Neovim Configuration

This is the Neovim vim Configuration that you can use.


## Prerequisites
Git, ripgrep, fzf, Vim ,Neovim 

## Installation
```bash
#This will clone neovim configuration file into ~/.config/nvim Directory.
git clone https://github.com/doyalsnehil/nvim.git ~/.config/nvim/ 
```

## Usage


```bash
#Start Neovim 
nvim
#Wait till Lazy is installing plugins
```
